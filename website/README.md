# Canvas-AI website

A static marketing site (landing page + privacy/terms/refund). Just HTML/CSS —
host it anywhere, free.

## Before you publish — find & replace
- **`BUY_URL`** in `index.html` → your LemonSqueezy checkout link.
- **`REPLACE_ME`** everywhere (email/domain, "Last updated" dates) in
  `index.html`, `privacy.html`, `terms.html`, `refund.html`.

## Preview locally
Open `index.html` in a browser, or:
```
python -m http.server -d website 8080   # then visit http://localhost:8080
```

## Deploy (free)
Pick one:
- **Cloudflare Pages** or **Netlify** — drag-and-drop the `website/` folder, or
  connect this repo and set the build output dir to `website`. Free HTTPS.
- **GitHub Pages** — push the repo, enable Pages, point it at `/website`.

## Custom domain (optional, ~$10/yr)
Buy a domain at Cloudflare or Namecheap, then add it in your host's dashboard.

## Assets
`assets/` holds the icon and promo images (generated by
`python marketing/make_images.py`). Replace `hero.png` with a real app
screenshot when you have one for an even better landing page.
